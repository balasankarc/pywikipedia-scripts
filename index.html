This repo contains the scripts i wrote or modified, mostly to automate things around Malayalam Wikisource.

  1. Wikisource-Extract_Text_from_index_pages .py is a script that can be used to extract text from index pages of a book.
  2. Wikisource_Contributors_List is a script to generate the list of contributors to a specific work.
